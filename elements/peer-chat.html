<peer-chat> ​

Usage ​

  1. If you haven't already, go through the setup for the module.

  2. Import the <peer-chat> element somewhere in the javascript side of your web-app like this:

js
import '@darksoil-studio/messenger-zome/dist/elements/peer-chat.js'
  1. Use it in the html side of your web-app like this:
html
<peer-chat .peerChatHash=${peerChatHash}>
</peer-chat>
html
<peer-chat peerChatHash={peerChatHash}>
</peer-chat>
html
<peer-chat [peerChatHash]="peerChatHash">
</peer-chat>
html
<peer-chat :peerChatHash="peerChatHash">
</peer-chat>
html
<peer-chat peerChatHash={peerChatHash}>
</peer-chat>

WARNING

Like all the elements in this module, <peer-chat> needs to be placed inside an initialized <messenger-context>.
