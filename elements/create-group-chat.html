Usage ​

  1. If you haven't already, go through the setup for the module.

  2. Import the <create-group-chat> element somewhere in the javascript side of your web-app like this:

js
import '@darksoil-studio/messenger-zome/dist/elements/create-group-chat.js'
  1. Use it in the html side of your web-app like this:
html
<create-group-chat>
</create-group-chat>

WARNING

Like all the elements in this module, <create-group-chat> needs to be placed inside an initialized <messenger-context>.
